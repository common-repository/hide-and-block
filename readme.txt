== Description ==

With this plugin you can hide the admin bar and dashboard for none admins. If someone try to access wp-admin will be redirected to home URL.



== Frequently Asked Questions ==

=How can i use it?=

Activate the plugin and admin bar and dashboard will just be visible for admins.
